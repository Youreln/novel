# 中梧小说网站 - ZhongWuNovel
📖 中梧小说开源纯净在线阅读平台，内置定时更新脚本，零广告阅读体验，支持个人私有化搭建

## 项目介绍
中梧小说是一套完整可独立部署的小说Web阅读系统，面向个人搭建自用阅读站点、开发者学习练手、自定义二次开发。
系统自带定时任务，可周期性更新小说章节内容；区分读者前台与管理员后台，布局适配电脑、手机、平板全设备。
代码完全开源透明，无捆绑插件、无强制广告。

## 技术栈
按需替换为你实际使用的框架：
- 后端：SpringBoot3 / NestJS / FastAPI
- 前端：Vue3 + Vite + TailwindCSS
- 数据库：MySQL / SQLite（低配机器首选）
- 定时任务：内置定时调度，支持自定义更新频率

## 核心功能
### 读者前台
- 小说分类筛选、全文模糊搜索、新书/热门榜单
- 多套阅读皮肤：日间、夜间护眼模式，字号、行间距自定义
- 个人书架收藏、阅读进度自动记忆、历史浏览记录
- 移动端完美适配，纯净无弹窗广告

### 管理后台
- 书籍信息编辑、封面上传、批量录入书籍
- 定时更新配置：自定义时段批量刷新小说章节，支持手动一键更新
- 用户信息管理、评论审核、站点基础参数设置
- 缓存清理、运行状态查看

### 项目特色
✅ 定时自动更新小说，减少人工维护成本
✅ 极简清爽UI，专注阅读本身
✅ 低配置服务器即可流畅运行
✅ 代码注释完整，结构清晰，便于二次修改拓展
✅ 支持本地自有文档书籍导入展示

## 快速部署教程
### 环境准备
1. 安装对应运行环境（JDK17 / Node18 / Python3.10+）
2. 导入项目根目录SQL数据库脚本
3. 修改配置文件内数据库地址、端口、定时任务参数

### 启动运行命令
```bash
# Java SpringBoot示例
java -jar zhongwu-novel.jar

# Node Nest示例
npm install
npm run start

# Python FastAPI示例
pip install -r requirements.txt
python main.py
